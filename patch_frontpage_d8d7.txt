Drupal VERSION8 and VERSION7 released

<p style="position: inherit;">Drupal VERSION8 and Drupal VERSION7, maintenance releases with numerous bug fixes (no security fixes), are now available for download. <!--break-->See the <a href="/drupal-VERSION8-release-notes">Drupal VERSION8 release notes</a> and <a href="/drupal-VERSION8-release-notes">Drupal VERSION8 release notes</a> for full lists of included fixes.</p>

<p style="position: inherit;"><strong><a href="/upgrade/">Upgrading</a> your existing Drupal 8 and 7 sites is recommended.</strong> There are no major nor non-backwards-compatible features in these releases. For more information about the Drupal 8.x release series, consult the <a href="https://www.drupal.org/8">Drupal 8 overview</a>.</p>

<div style="float: right; padding: 0 0 2em 1em; margin-left: 2em; width: 35%; border-left: 1px solid #cdcdcd;">

<h2>Security information</h2>

We have a <a href="/security">security announcement mailing list and a history of all security advisories</a>, as well as an <a href="/security/rss.xml">RSS feed with the most recent security advisories</a>. We <strong>strongly advise Drupal administrators to <a href="/security">sign up</a></strong> for the list.

Drupal 8 includes the built-in Update Manager module, which informs you about important updates to your modules and themes.

There are no security fixes in these releases of Drupal core.

<h2>Bug reports</h2>

Drupal 8.0.x and 7.x actively maintained, so more maintenance releases will be made available, according to our <a href="https://www.drupal.org/documentation/version-info#when">monthly release cycle</a>.
</div>

<h2>Change log</h2>

Drupal VERSION8 contains bug fixes and documentation and testing improvements only. The full list of changes between the last 8.0.x patch release and the VERSION8 release can be found by reading the <a href="/drupal-VERSION8-release-notes">VERSION8 release notes</a>. A complete list of all changes in the stable 8.0.x branch can be found in <a href="http://cgit.drupalcode.org/drupal/log/?id=refs/heads/8.0.x&nocache=VERSION8">the git commit log</a>.
