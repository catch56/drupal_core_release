Maintenance and security release of the Drupal 8 series.

This release fixes <strong>security vulnerabilities</strong>. Sites are <strong><a href="/node/1494290">urged to upgrade immediately</a></strong> after reading the notes below and the security announcements:
<ul>
<li><a href="/SA-CORE-SA_NUMBER">Drupal Core - Critical - Multiple Vulnerabilities - SA-CORE-SA_NUMBER</a></li>
</ul>

No other fixes are included.

No changes have been made to the .htaccess, web.config, robots.txt or default settings.php files in this release, so upgrading custom versions of those files is not necessary.

<h3>Known issues</h3>

See the list of the <a href="https://www.drupal.org/project/issues/drupal?version=BRANCH8-dev">known issues for the BRANCH8 branch</a>. There are no known regressions in this release.
