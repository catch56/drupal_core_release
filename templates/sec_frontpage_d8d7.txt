Drupal VERSION8 and VERSION7 released

<p style="position: inherit;">Drupal VERSION8 and Drupal VERSION7, maintenance releases which contain fixes for <strong>security vulnerabilities</strong>, are now available for download. <!--break-->See the <a href="https://www.drupal.org/drupal-VERSION8-release-notes">Drupal VERSION8</a> and <a href="https://www.drupal.org/drupal-VERSION7-release-notes">Drupal VERSION7</a> release notes for further information.</p>

<a href="http://ftp.drupal.org/files/projects/drupal-VERSION7.tar.gz" class="link-button"><span>Download Drupal VERSION7</span></a></div>

<p style="position: inherit;"><strong><a href="/upgrade/">Upgrading</a> your existing Drupal 8 and 7 sites is strongly recommended.</strong> There are no new features nor non-security-related bug fixes in these releases. For more information about the Drupal BRANCH8 release series, consult the <a href="https://www.drupal.org/8">Drupal 8 overview</a>. More information on the Drupal 7.x release series can be found in the <a href="https://www.drupal.org/drupal-7.0">Drupal 7.0 release announcement</a>.</p>

<div style="float: right; padding: 0 0 2em 1em; margin-left: 2em; width: 35%; border-left: 1px solid #cdcdcd;">

<h2>Security information</h2>

We have a <a href="/security">security announcement mailing list and a history of all security advisories</a>, as well as an <a href="/security/rss.xml">RSS feed with the most recent security advisories</a>.  We <strong>strongly advise Drupal administrators to <a href="/security">sign up</a></strong> for the list.

Drupal 8 and 7 include the built-in Update Manager module, which informs you about important updates to your modules and themes.

<h2>Bug reports</h2>

Both Drupal BRANCH8 and 7.x are being maintained, so given enough <a href="https://drupal.org/project/issues/search/drupal?status[]=8&categories[]=1">bug fixes</a> (not just <a href="https://drupal.org/project/issues/search/drupal?status[]=1&categories[]=1">bug reports</a>) more maintenance releases will be made available, according to our <a href="https://www.drupal.org/documentation/version-info#when">monthly release cycle</a>.

</div>

<h2>Change log</h2>

Drupal VERSION8 is a security release only. For more details, see the <a href="https://www.drupal.org/drupal-VERSION8-release-notes">VERSION8 release notes</a>.  A complete list of all changes in the stable BRANCH8 branch can be found in <a href="http://cgit.drupalcode.org/drupal/log/?h=BRANCH8">the git commit log</a>.

Drupal VERSION7 is a security release only. For more details, see the <a href="https://www.drupal.org/drupal-VERSION7-release-notes">VERSION7 release notes</a>.  A complete list of all changes in the stable 7.x branch can be found in <a href="http://cgit.drupalcode.org/drupal/log/?h=7.x">the git commit log</a>.

<h2>Security vulnerabilities</h2>

Drupal VERSION8 and VERSION7 were released in response to the discovery of security vulnerabilities. Details can be found in the official security advisories:

<ul>
<li><a href="/SA-CORE-SA_NUMBER">Drupal Core - Critical - Multiple Vulnerabilities - SA-CORE-SA_NUMBER</a></li>
</ul>

To fix the security problem, please upgrade to either Drupal VERSION8 or Drupal VERSION7.

<h2>Update notes</h2>

See the <a href="http://drupal.org/drupal-VERSION8-release-notes">VERSION8</a> and <a href="http://drupal.org/drupal-VERSION7-release-notes">VERSION7</a> release notes for details on important changes in this release.

See the <a href="https://www.drupal.org/drupal-VERSION8-release-notes">VERSION8 release notes</a> or <a href="https://www.drupal.org/drupal-VERSION7-release-notes">VERSION7 release notes</a> for a list of known issues affecting each release.
