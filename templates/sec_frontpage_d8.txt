Drupal VERSION8 released

<p style="position: inherit;">Drupal VERSION8, a maintenance release which contains fixes for <strong>security vulnerabilities</strong>, is now available for download. <!--break-->See the <a href="https://www.drupal.org/project/drupal/releases/VERSION8">Drupal VERSION8</a> release notes for further information.</p>

<div  style="text align: center; margin: 2em;"><a href="http://ftp.drupal.org/files/projects/drupal-VERSION8.tar.gz" class="link-button" style="margin: 0 0 0.6em 0;"><span>Download Drupal VERSION8</span></a></div>

<p style="position: inherit;"><strong><a href="/upgrade/">Upgrading</a> your existing Drupal 8 sites is strongly recommended.</strong> There are no new features nor non-security-related bug fixes in this release. For more information about the Drupal BRANCH8 release series, consult the <a href="https://www.drupal.org/8">Drupal 8 overview</a>.

<div style="float: right; padding: 0 0 2em 1em; margin-left: 2em; width: 35%; border-left: 1px solid #cdcdcd;">

<h2>Security information</h2>

We have a <a href="/security">security announcement mailing list and a history of all security advisories</a>, as well as an <a href="/security/rss.xml">RSS feed with the most recent security advisories</a>.  We <strong>strongly advise Drupal administrators to <a href="/security">sign up</a></strong> for the list.

Drupal 8 includes the built-in Update Manager module, which informs you about important updates to your modules and themes.

<h2>Bug reports</h2>


Drupal BRANCH8 is actively maintained, so more maintenance releases will be made available, according to our <a href="https://www.drupal.org/documentation/version-info#when">monthly release cycle</a>.
</div>

<h2>Change log</h2>

Drupal VERSION8 is a security release only. For more details, see the <a href="https://www.drupal.org/project/drupal/releases/VERSION8">VERSION8 release notes</a>.  A complete list of all changes in the stable BRANCH8 branch can be found in <a href="http://cgit.drupalcode.org/drupal/log/?h=BRANCH8">the git commit log</a>.

<h2>Security vulnerabilities</h2>

Drupal VERSION8 was released in response to the discovery of security vulnerabilities. Details can be found in the official security advisories:

<ul>
<li><a href="/SA-CORE-SA_NUMBER">Drupal Core - Critical - Multiple Vulnerabilities - SA-CORE-SA_NUMBER</a></li>
</ul>

To fix the security problem, please upgrade to Drupal VERSION8.
